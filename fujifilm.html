Fujfilm Dimatix: Software Engineering Intern

May 2021 - August 2021

I used OpenCV to create printer head analysis software that automatically processes images and analyzes them for the relevant information. The program then generates a heatmap of the printer heads, allowing users to more easily see the condition of the printer heads in the uploaded images. This data is then uploaded alongside other relevant values into a report which does not contain the relevant information and shows the step-by-step analysis allowing users to find any potential errors that could show up.
