Tillandsia Wind Turbine

A completely custom wind turbine designed to be hung outside your window in order to generate power. It is currently in development

Roombas Per Minute

An combat robot made for the Combat Robotics Hackathon at WPI. It features a shell spinner design in order to increase its durability as well as the damage it can output. In combat, however, there was a flaw where if it was hit from below its inertia would cause the robot to flip and be wiped out.

RBE 3002
Final Project

I developed a ROS program using SLAM, AMCL, and A star to enable a Turtlebot 3 Burger to autonomously navigate, map, and localize inside a maze. C-space, frontier detection, navigation, and frontier detection algorithms were made from scratch. The nodes were programmed in Python, and the launch files were custom.

RBE 3001
Final Project

I developed a program to perceive, grasp, and sort colored objects using a robot arm and a camera, using MATLAB.

RBE 2002
Final Project

I used C++, computer vision, MQTT protocol, and various sensors and transmitters to program 3 robots to perform scenes from Romeo and Juliet. Additionally each robot was customized to perform its role.

RBE 2001 Final Project
primis vestibulum
