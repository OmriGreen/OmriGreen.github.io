Fabric Microfulfillment: Software Engineering Intern

May 2021 - August 2021

I used python to create a data analysis tool for analyzing robot fleet data. It was designed to be modular, working with the current variety of robots and future robots. The software worked in the following manner: A user would upload a csv file containing the unprocessed data from the robot fleet, then after pressing a button the code would generate a report showing how many errors occurred , the mean time between failure both for individual errors and for all errors, where each error occurs, as well as the errors over time. All of this is done for both every kind of robot as well as for every kind of robot or even an individual robot or type of robot depending on the specified analysis inputs. A user can also specify which errors to analyze, for example a user can choose to analyze where the robots lose contact with the central server to find dead zones in the warehouses.
