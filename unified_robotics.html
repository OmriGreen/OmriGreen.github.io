Robotic Replacement of Tilted Panels

I designed a robot-mounted 4-bar mechanism to position panels on tilted surfaces at various heights and locations using the SolidWorks design suite.

I also created two different grippers and mounted them on two separate robots to complete the pick-and-place tasks.

Thus With a Kiss I Die

I collaborated with a team to implement a multi-robot system featuring three robots: Romi, Tybalt/Juliet, and Mercutio/Friar Lawrence (with some robots taking on dual roles). Inter-robot communication relied on the MQTT protocol over WiFi, ensuring reliable coordination of cues and timing for the scenes in Romeo and Juliet. This approach proved more robust than alternatives like IR emitters and receivers, which suffered from range and visibility issues. The robots also used AprilTag tracking with an OpenMV camera, enabling consistent following behavior. Each character was assigned a “script” containing scene-specific tasks, and upon receiving a trigger, the robot advanced to the next programmed action. All robots also shared a common base program to handle general functions, since they used the same drive platform.

Robotic Pick and Place System

This project involved building a vision-guided robotic arm system that used quintic trajectory planning for motion control. Test trials showed the robot could detect and localize balls, then sort them by color using the planner. The system also supported dynamic object tracking and manipulation of unique objects.

Automated Maze Mapping and Exploration

This project focused on developing a ROS-based program that enabled a TurtleBot 3 Burger to autonomously map, navigate, and localize within a maze using SLAM, AMCL, and A*. My team implemented C-space generation, frontier detection, and navigation algorithms from scratch. All nodes were programmed in Python, with custom launch files created for the system.
